Creative Computing


Homework #2 - HTML Story

Create a webpage using HTML that tells a story or describes a process.


Requirements

  • Your code must have at least 3 distinct pages.
  • Your code must use the <a> tag at least once on each page to link to the next page (or back to the beginning).
  • Your code must use the style attribute at least once on each page to position an element.

References

  • Maeda @ Media - John Maeda
  • Design By Numbers - John Maeda
  • Choose Your Own Adventure

Submission

You will submit your homework by uploading a zip file of all of the html files and any assets used by your page to canvas.


Things To Think About

  • How can you take advantage of the text placement to enhance the experience?
  • How will your page look on different devices? Or in a browser with different dimensions?
  • Is there a way to make the experience of moving between pages more meaningful?
