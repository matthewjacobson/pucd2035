Homework #4 - Josef Albers Animated

Recreate one of the images from Josef Albers Interaction of Color and animate it using css transforms and/or keyframes.


Requirements

  • Your submission must include one html file and one css file.
  • Your page should use one of the images from Josef Albers Interaction of Color as a starting point.
  • Your page should be animated.

References

  • Josef Albers - Interaction of Color
  • Josef Albers - The Logic and Magic of Color

Submission

You will submit your homework by uploading a zip file that includes the html and css files to canvas.
